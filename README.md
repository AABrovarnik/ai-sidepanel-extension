# AI Chrome Side Panel Assistant

## Features
- Chat with AI
- Summarize pages
- Explain selected text
- Multi-provider (OpenAI, Ollama)
- Streaming responses

## Setup
1. Add API key
2. Load extension in Chrome
3. Use shortcuts

## Shortcuts
- Ctrl+Shift+Y — open panel
- Ctrl+Shift+S — summarize
- Ctrl+Shift+E — explain selection
